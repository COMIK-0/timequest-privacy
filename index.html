Политика конфиденциальности TimeQuest

Дата вступления в силу: 15 июня 2026 года

TimeQuest — приложение для личного планирования задач, отслеживания прогресса и получения рекомендаций по организации времени.

Какие данные обрабатывает приложение

TimeQuest может сохранять на устройстве названия и описания задач; категории, приоритеты, сложность, сроки и статусы задач; планы дня, прогресс, XP, уровни, достижения и серии дней; настройки приложения, уведомлений, темы и фокус-сессии.

Для работы приложения не требуется регистрация или создание пользовательского аккаунта.

Локальное хранение

Основные пользовательские данные хранятся локально на устройстве в базе данных приложения и не отправляются на собственный сервер разработчика.

Android может включать данные приложения в системную резервную копию, если резервное копирование включено пользователем. Управление такой копией выполняется средствами операционной системы и выбранного пользователем поставщика резервного копирования.

Персональный ИИ-разбор

Функция персонального ИИ-разбора является дополнительной и запускается только по действию пользователя после получения согласия.

Для формирования совета в Firebase AI Logic передаётся агрегированная статистика: количество активных, выполненных, пропущенных и отменённых задач; общие показатели выполнения; XP, активные дни, этап использования приложения и тенденции за период.

Названия и описания задач не передаются для формирования ИИ-совета.

Обработка запросов выполняется сервисами Firebase и Google в соответствии с их условиями и политиками конфиденциальности. При использовании функции сервисы могут обрабатывать технические данные, необходимые для выполнения запроса.

Обычные функции планирования, локальная статистика и сохранённые данные работают без использования ИИ-функции и без подключения к интернету.

Разрешения устройства

  • Доступ к интернету — для дополнительной функции Firebase AI Logic.
  • Разрешение на уведомления — для локальных напоминаний.
  • Разрешение на планирование точных будильников — для своевременных локальных напоминаний.
  • Получение события загрузки устройства — для восстановления уведомлений после перезагрузки.

Передача данных третьим лицам

Разработчик не продаёт пользовательские данные и не передаёт их рекламным сервисам. При использовании персонального ИИ-разбора агрегированные данные обрабатываются сервисами Firebase и Google.

Удаление данных

Поскольку приложение не использует пользовательские аккаунты и собственный сервер, удалить локальные данные можно, очистив данные TimeQuest в системных настройках Android или удалив приложение. Системными резервными копиями необходимо управлять через настройки соответствующего сервиса резервного копирования.

Изменения политики

Политика может обновляться при изменении функций приложения или требований законодательства. Актуальная версия публикуется по той же ссылке.
